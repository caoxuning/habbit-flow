# HabitFlow FastAPI 后端已完成功能

## 1. 后端目录

已新增独立 Python 后端目录：

```text
backend_fastapi
```

该目录与原 Java 后端 `backend` 相互独立，不会影响原有 Spring Boot 项目。

## 2. 技术栈

已采用：

- Python
- FastAPI
- SQLAlchemy
- MySQL
- JWT
- Passlib BCrypt
- Uvicorn

## 3. 已完成基础功能

### 3.1 认证模块

已完成：

- 用户注册
- 用户登录
- 密码加密存储
- JWT 登录令牌生成
- 登录状态校验

接口：

```text
POST /api/auth/register
POST /api/auth/login
```

### 3.2 用户模块

已完成：

- 获取个人资料
- 修改个人资料
- 修改密码
- 原密码校验
- 用户名重复校验

接口：

```text
GET /api/user/profile
PUT /api/user/profile
```

### 3.3 目标管理模块

已完成：

- 创建目标
- 查询目标列表

支持字段：

- 目标名称
- 目标类型
- 开始日期
- 结束日期
- 目标周期
- 每日目标次数
- 目标状态

### 3.4 打卡模块

已完成：

- 今日打卡

- 打卡记录列表
- 按目标筛选打卡记录
- 防止重复打卡

- 校验打卡日期是否在目标周期内

接口：

```text
GET /api/check-ins
```

### 3.5 统计模块

## 5. 启动方式

进入 Python 后端目录：

```bash
cd backend_fastapi
```

创建虚拟环境：

```bash
py -m venv .venv
```

激活虚拟环境：

```bash
.venv\Scripts\activate
```

安装依赖：

```bash
pip install -r requirements.txt
```

启动服务：

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8081
```

接口文档地址：

```text
http://localhost:8081/docs
```

## 6. 数据库配置

默认连接：

```text
mysql+pymysql://root:root@localhost:3306/habitflow?charset=utf8mb4
```

可通过环境变量修改：

```bash
set DATABASE_URL=mysql+pymysql://用户名:密码@localhost:3306/habitflow?charset=utf8mb4
```
