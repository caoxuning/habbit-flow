# HabitFlow 项目启动指南

> 前后端分离架构 | 前端 Vue3 + Element Plus | 后端 FastAPI + MySQL

---

## 前置条件

| 依赖 | 版本要求 | 说明 |
|:---|:---:|:---|
| Node.js | v16+ | 运行前端 |
| Python | 3.10+ | 运行后端 |
| MySQL | 8.0+ | 云数据库（阿里云 RDS） |

---

## 启动步骤

### 1️⃣ 启动后端 (FastAPI)

```bash
# 进入后端目录
cd backend_fastapi

# 激活虚拟环境（Windows）
.venv\Scripts\activate

# 启动服务（端口 8081）
python -m uvicorn app.main:app --host 0.0.0.0 --port 8081 --reload
```

- ✅ 看到 `Uvicorn running on http://0.0.0.0:8081` 即为成功
- 🌐 API 文档：http://localhost:8081/docs (Swagger UI)
- 🔄 `--reload` 表示修改代码后自动重启

### 2️⃣ 启动前端 (Vite)

```bash
# 进入前端目录
cd frontend

# 启动开发服务器（默认端口 5173）
npm run dev
```

- ✅ 看到 `Local: http://localhost:5173/` 即为成功
- 🚀 打开浏览器访问 http://localhost:5173
- 🔄 Vite 自动将 `/api` 请求代理到后端 `http://localhost:8081`

### 3️⃣ 验证启动

打开浏览器访问 http://localhost:5173，应看到登录页面。
若 5173 端口被占用，Vite 会自动切换到 5174，查看终端输出提示。

## 使用流程

```
1. 注册账号 → 2. 创建目标 → 3. 打卡 → 4. 查看成长日志 → 5. 导出数据
```

| 步骤 | 操作 | 位置 |
|:---:|:---|:---|
| ① | 注册新账号（输入用户名+密码+邮箱） | 登录页切换「注册」标签 |
| ② | 创建目标（如：类型填"运动"，周期选"每日"） | 左侧菜单「目标管理」→ 新建目标 |
| ③ | 打卡（选择目标，点击今日打卡） | 「目标管理」表格打卡按钮 / 「打卡记录」页面 |
| ④ | 查看打卡成功后弹出的精选内容 | 打卡后自动弹出 ElMessageBox |
| ⑤ | 查看成长日志时间线 | 左侧菜单「成长日志」(Timer图标) |
| ⑥ | 导出打卡记录为 Excel | 「打卡记录」页面顶部工具栏 |
| ⑦ | 查看数据图表和统计 | 「数据概览」仪表盘 |

---

## 新增功能（本次实现）

### ① 内容推荐系统
- **触发**：每次打卡成功后自动弹窗
- **内容**：名人名言（含中文翻译）、英语词汇（含例句）、学习技巧
- **补充**：同路人的打卡寄语

### ② 成长日志时间线
- **位置**：左侧菜单「成长日志」（⏱ Timer 图标）
- **展示**：日期倒序，包含打卡记录、勋章获得、每日推荐
- **筛选**：支持 7天/30天/90天/全部

### ③ 数据导出
- **位置**：「打卡记录」页面顶部工具栏
- **格式**：Excel (.xlsx)，含日期、目标、备注、类型、时间

## 项目结构

```
habitflow/
├── frontend/                    # 前端 (Vue3 + Element Plus + ECharts)
│   ├── src/
│   │   ├── App.vue             # 主页面（所有视图和逻辑）
│   │   ├── main.js             # Vue 入口
│   │   ├── style.css           # 全局样式
│   │   └── api/
│   │       ├── http.js         # Axios（拦截器、代理配置）
│   │       └── index.js        # 所有 API 封装
│   └── vite.config.js          # Vite 配置（代理到 8081）
│
├── backend_fastapi/             # 后端 (FastAPI + SQLAlchemy)
│   └── app/
│       ├── main.py             # 入口，注册所有路由
│       ├── models.py           # 13 张表 ORM 模型
│       ├── schemas.py          # Pydantic 请求/响应模型
│       ├── business.py         # 勋章发放、进度计算等核心逻辑
│       ├── inspirations.py     # 内容推荐系统
│       ├── security.py         # JWT 生成/解析
│       ├── common.py           # 异常处理、响应格式
│       ├── database.py         # 数据库连接
│       ├── deps.py             # 依赖注入（获取用户、数据库会话）
│       └── routers/
│           ├── auth.py         # 登录注册
│           ├── users.py        # 用户信息
│           ├── goals.py        # 目标 CRUD
│           ├── checkins.py     # 打卡/补卡
│           ├── stats.py        # 仪表盘/时间线
│           ├── badges.py       # 勋章查询
│           ├── exports.py      # Excel 导出
│           └── ...             # 社交路由（部分实现）
│
├── database/
│   └── schema.sql              # 完整建表 SQL（已更新）
│
└── docs/
    ├── 启动项目指南.md           # ← 本文档
    ├── 数据库设计文档.md         # 13张表详细设计
    └── 功能清单文档.md          # 全部功能清单
```
